# Packaging a Python application with Docker

**Cloud Computing and Distributed Systems**
Università degli Studi di Verona — A.A. 2025/2026

---

## What you will do

In this exercise you will package a small Python web application into a Docker image. You will then observe how Docker's layer cache works in practice, and apply a multi-stage build to reduce the final image size.

You do not need to understand the Python code — treat it as a black box. Your focus is entirely on the Dockerfile and the Docker commands.

All the files you need are already in the `/root/app` directory.

---

## How to follow this exercise

Each step contains:
- a brief explanation of the concept
- the exact commands to run — copy and paste them into the terminal
- a question to answer before moving to the next step

Take your time to read the output carefully before answering each question.
