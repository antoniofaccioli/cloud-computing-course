# IoT Sensor Registry — MongoDB Autonomous

This is the autonomous version of the sensor registry exercise. The scenario and dataset are identical to the guided version — the complete commands are not provided.

Each step gives you:
- The **objective** — what you need to achieve
- The **data** to work with (where applicable)
- The **expected output** to verify your result
- A **commands to use** reference — the names of the relevant commands and operators, without syntax

The database name is `sensor_registry` and the collection is `sensors`.
The MongoDB 7 image has already been pulled.
