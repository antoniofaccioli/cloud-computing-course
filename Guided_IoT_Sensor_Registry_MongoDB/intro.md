# Building an IoT Sensor Registry with MongoDB

In this exercise you will build and manage a MongoDB collection that represents a fleet of IoT sensors deployed across a facility.

You will practice:
- Inserting documents with nested objects and array fields
- Querying with comparison and membership operators (`$gt`, `$in`, direct array match)
- Updating individual and multiple documents with `$set` and `$push`
- Running a multi-stage aggregation pipeline (`$match`, `$group`, `$sort`)
- Deleting documents and verifying the result

The MongoDB 7 image and all script files have been prepared during setup.
