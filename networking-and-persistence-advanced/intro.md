## How this exercise works

This exercise has three objectives. For each one you are told **what** to achieve — not how. Use the Docker knowledge from the lecture and the demo to reach each goal independently.

All the files you need are in `/root/app`:
- `counter.py` — a Flask application that counts visits and writes the count to `/data/counter.json`
- `requirements.txt` — its dependencies
- `Dockerfile` — already written and ready to build

If you completed Exercise A first, you already know the application. If this is your starting point, build the image and explore the behaviour before attempting each objective.
