## How this exercise works

This exercise has three objectives. For each one you are told **what** to achieve, not **how** to do it. Use the Docker commands and concepts from the lecture and the demo scenario.

All the files you need are in `/root/app`:
- `app.py` — the application
- `requirements.txt` — its dependencies
- `Dockerfile.v1`, `Dockerfile.v2`, `Dockerfile.v3` — three Dockerfiles to work with
