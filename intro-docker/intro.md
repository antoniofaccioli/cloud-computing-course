**Cloud Computing and Distributed Systems**

## Welcome to Killercoda

This is a real Linux environment running in your browser. No installation required. The terminal on the right executes commands on a real Ubuntu machine with Docker already installed.

The environment stays active for **60 minutes** from when you opened it. Everything is lost when the session ends.

---

## What this scenario covers

This scenario introduces the Docker commands you will need in the exercises that follow. The goal is not the application — it is the commands and how Docker behaves.

You will learn to:
- check that Docker is available and explore its CLI
- build an image from a Dockerfile
- run, inspect, and stop a container
- observe Docker's layer cache in action
